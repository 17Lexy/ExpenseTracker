# Expense Tracker Bot
### Твой личный помощник в учете расходов



Этот телеграм бот быстро и удобно поможет зафиксировать траты на автомобиль, показать историю и подсчитать потраченную сумму. 

#### Стек проекта:
   
* ***Java 21***
 
* ***Spring***

* ***PostgreSQL***
